Getting Started

Getting started with this application really means getting started with a hardware robot. Reference should be made to hardware specific help files to get a particular robot set up. Working with the RoboConsole probably comes after you have the little fellow charged and running about and want to get to grips on a deeper level. There are two fundamental ways to engage with the application:

  • As the user of an existing robot extension.
  • As a developer of the system.

As a user of the system you'll need a set of application plugins to support your hardware. At the time of writing the only robot supported is the Surveyor SRV1. This is a well made little machine created by the Surveyor Corporation and packing some fairly powerful vision technology. Really its a platform for experiments with robotic vision. The SRV1 specific help files can add a little colour and detail.

As a developer there are many ways to engage at a code level:

  • Adding small extensions to an existing robot plugin set
  • Adding a new type of robot by creating your own set of plugins
  • Joining in the open source effort to develop the application itself

In this Getting started set of help files we'll give an overview of all of these means of engagement.
