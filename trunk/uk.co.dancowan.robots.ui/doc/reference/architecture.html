Overview

The RoboWorkbench is built as an Eclipse RCP application. This allows for the rapid development of a modular and extensible application using OSGi and the plugin framework. Some familiarity with the Eclipse RCP will be assumed here and this overview will focus on the robot specific components and extensions.

I will refer the SRV1 plugin (uk.co.dancowan.robots.srv) as an example of extending the base RoboWorkbench system to add an actual robot. Beneith this plugin lie the two main plugins which support the workbench itself: the HAL plugin (uk.co.dancowan.robots.hal) and the UI plugin (uk.co.dancowan.robots.ui).
